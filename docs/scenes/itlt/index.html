This project was originally created as the capstone for my Creative Writing degree at Oberlin college. It has turned into an interactive web art piece that I hope will give you a small demonstration of the power of virtual spaces to create a sort of "anti-reality", an experience hopefully as playful, interesting, and "real" as any other.
